```
./configure
make
```

/y/izuchenie_stat_analizatorov/bftpd/src

# Создаём директории

~/bftpd-test (пустую, возможно она не нужна), 

# Правим конфиг

В следующей строчке
```
group list,den73,@games,@users {
```
нужно написать два существующих пользователя и две существующих группы.

# Команда для запуска

cd /y/izuchenie_stat_analizatorov/bftpd/src
sudo ./bftpd -d -c /y/izuchenie_stat_analizatorov/bftpd/run/bftpd.conf

Далее подключаемся ftp 127.0.0.1 и вводим логин и пароль существующего пользователя. 
Успешный логин означает, что программа нигде не упала (она собрана с санитайзером, который
отлавливает некоторые ошибки, например, переполнение буфера и чтение неинициализированных
данных).
